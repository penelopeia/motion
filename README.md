Playing with Camera Motion
==========================

## Description

Try out some creative coding with python and openCV

## Getting Started

### Requirements :key:

* Python 3.6+
* [opencv-python](https://pypi.org/project/opencv-python/)
* camera/webcam

### Installing :floppy_disk:

* pip install -r requirements.txt

### Usage :computer:

* python main.py
